# 🚀 Linux Cloud Automation: Secure Nginx Deployment
**Author:** Sonal Solanki (Aspiring Cloud Engineer)

## 📌 Project Overview
This repository contains a professional Bash automation script designed to streamline the deployment of a secure web portal on a Linux (CentOS/Ubuntu) environment. 

## 🛠️ Technical Features
* **Automated Directory Structuring**: Programmatically creates secure web roots.
* **Symbolic Link Management**: Uses `ln -sf` to map custom project folders to Nginx web paths.
* **Service Orchestration**: Automatically reloads the Nginx service to apply configuration changes.
* **Security Best Practices**: Includes execution permission handling (`chmod +x`).

## 💻 Skills Demonstrated
* **Linux Administration**: Deep understanding of the filesystem and permissions.
* **Shell Scripting**: Automation of repetitive sysadmin tasks.
* **Version Control**: Mastery of the Git workflow (Stage, Commit, Push) using secure Personal Access Tokens.

## 🚀 How to Run
1. Clone the repository: `git clone https://github.com/sonalsolanki3112-pixel/Linux-Cloud-Automation.git`
2. Give execution rights: `chmod +x setup_site.sh`
3. Run the script: `./setup_site.sh`
